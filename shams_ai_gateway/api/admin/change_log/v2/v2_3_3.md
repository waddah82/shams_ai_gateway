## Version 2.3.3

### Fixes
- **PaddleOCR dependencies made optional** — PaddlePaddle lacks Python 3.14 wheels, which blocked installation on Frappe v16. PaddleOCR is now an optional extra (`pip install frappe-assistant-core[ocr]`). Users without it get a clear error message guiding them to install the extra or switch to Ollama vision backend.

### Improvements
- **CI now tests on Frappe v15 and v16** — full test suite runs on both Python 3.12 (v15) and Python 3.14 (v16) to catch compatibility issues before release
- **Automated releases with semantic-release** — version bumping, tagging, and GitHub Release creation are now fully automated using the same toolchain as Frappe and ERPNext
- **Conventional commit enforcement** — commitlint validates commit message format on all PRs
