## Version 2.3.2

### New — OCR & Document Extraction

- **PaddleOCR integration** — extract text from images and scanned PDFs using PaddleOCR with support for multiple languages
- **Vision model support** — use Ollama-based vision models (e.g., deepseek-ocr) as an alternative OCR backend
- **Configurable OCR settings** — choose your preferred backend, language, timeout, and memory limits from Shams AI Gateway Settings

### Improvements

- **Audit log cleanup** — automatic retention policy (default 180 days) keeps your audit logs manageable without manual intervention
- **Better report feedback** — report tools now provide clear guidance when used incorrectly, instead of failing silently
- **OAuth localhost support** — native apps like OpenCode and MCP Inspector can now register with HTTP localhost redirect URIs without errors
- **MCP 2025-06-18 spec** — updated to the latest MCP authentication specification
- **Code execution limits** — configurable timeout, memory, and CPU limits for the Python sandbox to prevent runaway scripts
- **Settings defaults** — all settings now correctly apply their default values on fresh installs and upgrades

### Security Fixes

- **Fixed** unauthorized access to protected API endpoints
- **Fixed** sensitive credential values appearing in server logs
- **Removed** unused API endpoints that were unnecessarily exposed
