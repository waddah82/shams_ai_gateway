## Version 2.4.1

### Fixes
- **HRMS Skill link corruption (#153)** — Removed the v2.4.0 `rename_skill_to_sag_skill` migration that, on sites with HRMS installed, renamed HRMS's own `Skill` DocType to `SAG Skill`. The rename rewrote every Link field pointing at HRMS `Skill` (Employee Skill, Skill Assessment, Designation Skill, Expected Skill Set) and moved any existing HRMS Skill records into `tabFAC Skill`. A new repair patch reverts those Link references and recovers stranded HRMS records back into `tabSkill`. Upgrade from v2.4.0 → v2.4.1 heals affected sites automatically.
